## KeyGrunt

#### Desc:
* Grunt和 Grunt 插件是通过 npm 安装并管理的，npm是 Node.js 的包管理器。
  你可以认为 grunt 是 npm 下的一个包，而众多的 grunt 插件也是 npm 下的包，而 grunt 是用来结合 grunt 的插件们一起完成一些特定的 grunt 任务。
* 它是一个工具集框架，靠着各些插件来拓展其功能。

#### AimTo:
* 自动化，去除重复的劳作
