How To Use

    Copy Documents:

  • assets
  • grunt
  • libs
  • scripts
  • styles
  • node_modules(if has)

    Copy Files:

  • gruntfile.js
  • index.html
  • package.json (Change it for your true project name and git repository)

    The commands:

  • npm install install the necessary npm package
  • grunt run the grunt default tasks
  • grunt dev run the grunt dev tasks
  • grunt pro run the grunt production tasks
  • grunt test run the grunt test tasks
